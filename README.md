# Product Marketplace - Full Stack Application

A complete full-stack web application for posting and browsing products, built with HTML, CSS, JavaScript, Node.js, Express, and PostgreSQL.

## Features
- **Post Products**: Add new products with name, description, price, category, and image
- **Browse Products**: View all products with filtering and sorting options
- **Responsive Design**: Works on all devices (mobile, tablet, desktop)
- **RESTful API**: Complete backend API for product management
- **Database**: PostgreSQL for reliable data storage
